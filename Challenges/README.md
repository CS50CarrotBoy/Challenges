Required Elements

At least two sprites, with at least one sprite that is not the default cat.

At least three scripts across your project.

At least one variable to store and update information (such as score, health, timer, level, points, or another value relevant to your project).

At least one conditional statement (if, if-else) that causes different actions to occur based on a condition.

At least one loop (forever, repeat, repeat until) to perform actions repeatedly.

At least one custom block created using Make a Block, with at least one input parameter. Use this block to simplify or organize repeated actions in your project.



Design Recommendations



Break large tasks into smaller scripts whenever possible.

Use meaningful names for variables and custom blocks.

Apply abstraction by moving repeated code into custom blocks.

Test your project frequently as you build it.

Save your work regularly.



Additional Notes

Focus on creating a project that is both functional and enjoyable.

Ensure that all required programming concepts are clearly demonstrated in your final submission.
